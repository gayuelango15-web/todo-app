## To-Do App
A simple task management application build using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks
- Local Storage support
- Task statistics
- Responsive design

## Technoloies Used
- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project
2. Open todo.html in a browser
3. Start managing tasks

## Future Improvements
- Edit tasks
- Due dates
- Dark mode
- Task categories
